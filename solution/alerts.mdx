---
title: "Alerts"
---

The **myConnectedFleet platform** can trigger an alert when specific conditions are met.  Alerts can be set for a single vehicle or for the vehicles in a group. 

The available alert types are:

| **Name** | **Type** | **Description** |
| :------------- | :------------- | :---------------------------------- |
| Arrive/Leave location alert | arriveOrLeaveLocationAlert | Alert if a vehicle arrives at or leaves a specific location. |
| Camera event alert | a cameraEventAlert | Alert when a High-G Event or Emergency button press is received from a camera. |
| Cold chain alert | ? | Alert if a temperature is outside a defined threshold / if the refrigeration alarm is activated. |
| xxx | a driveTimeAlert | Alert if a vehicle is moving for longer than a defined threshold. |
| Drive without key alert | drivingWithoutAuthenticationKeyAlert | Alert if a vehicle is driven without an ID. |
| Exception alert | exceptionAlert | Alert if a vehicle performs a specific activity, for example idles longer than 20 minutes. |
| Fuel level alert | ? | Alert if fuel level is drop after a stop. |
| xxx | genericInputAlert | An alert if a temperature value is outside a defined threshold / if the refrigeration alarm is activated. |
| Idling alert | idlingAssetAlert | An alert if someone idles too long. |
| Input/PTO alert | inputAlert | An alert when an input or PTO is used. |
| Late start alert | lateStartAlert | An alert if a vehicle doesn’t start its day by a specific time. |
| Long drive time alert | ? | Create an alert if a vehicle is moving for longer than a defined threshold. Moving time includes driving and idling |
| Low battery alert | lowBatteryAlert | An alert when the ATU's Battery is low. |
| xxx | lowEVChargeAlert | An alert when the EV charge is low. |
| xxx | maintenanceFaultAlert | An alert if a maintenance fault is detected on a vehicle. |
| Late job alert | onTimeLateJobArrivalAlert | An alert if a vehicle does NOT arrive at a scheduled job on time.|                                                           
| Out of hours alert | outOfHoursAlert | An alert if a vehicle is used outside of working hours. |
| xxx | Posted Speed Alert | This alert notifies you when selected mobiles exceed posted speed limit by more than user defined maximum speed tolerance. Vehicle type will be taken into account when determining the limit.|
| Speed over X alert | speedOverXAlert  An alert if a vehicle exceeds a user-defined speed threshold. |
| Stop time alert | stopTimeAlert | An alert if a vehicle exceeds a user-defined stop time threshold |
| Tamper event alert | tamperEventAlert | An alert when a device is unplugged from a vehicle (M210 only). |
| Trailer rollover alert (VDC) | ? | Create an alert when a trailers rollover prevention system (VDC) detects and corrects lateral acceleration. | 
|Trailer equipment disconnected alert | ? | Create an alert when the power supply is not correctly connected to MCF equipment in the trailer and requires checking. This can indicate a problem with the power supply to the EBS / trailer, depending on the connection method  |



When the alert is triggered, the following can be set to receive the alert notification:
- One or more **myConnectedFleet** portal users
- One or more email addresses
- One or more mobile telephone numbers
