MICHELIN Connected Fleet provides a set of REST APIs to allow you to integrate MICHELIN Connected Fleet into virtually any platform. The APIs cover all the features in MICHELIN Connected Fleet, from telematics, driver behaviour, and vehicle management.

To make a call to the MICHELIN Connected Fleet REST API, you need an API **username**, **password** and your **customer accout number**.  Please speak to your account manager if you do not have these details.

MICHELIN Connected Fleet REST API uses **Basic** HTTP authentication. Include your username and password encoded in base64 in the `Authorization` header of your HTTP request, with your **customer accout identifier** included in the URL.

```bash
curl --request GET \
  --url https://api.masternautconnect.com/connect-webservices/services/public/v1/customer/{customerId}/tracking/live \
  --header 'Authorization: Basic <encoded-value>'
```
