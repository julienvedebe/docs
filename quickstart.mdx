My Connected Fleet provides a set of REST APIs to allow you to integrate My Connected Fleet into virtually any platform. The APIs cover all the features in My Connected Fleet, from telematics, driver behaviour, and vehicle management. 

To make a call to the My Connected Fleet REST API, you need an API **username**, **password **and your **My Connected Fleet accout number**.  Please speak to your My Connected Fleet account manager if you donot have these details.

My Connected Fleet REST API uses **Basic **HTTP authentication. Include your username/password encoded in base64 `Authorization` header of your HTTP request:
